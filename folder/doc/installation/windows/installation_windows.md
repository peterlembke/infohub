# Install on Windows

## Docker
If you want to run docker then you need to do these steps on Windows.

Install [Docker desktop for Windows](https://docs.docker.com/desktop/windows/install/)
Then continue [here](/folder/doc/installation/docker/installation_docker.md).

## Vagrant + VirtualBox
You can run Vagrant + VirtualBox.
Install [VirtualBox](https://www.virtualbox.org/wiki/Downloads) and [Vagrant](https://www.vagrantup.com/downloads).
Then continue [here](/vagrant/README.md)

## MAMP / MAMP PRO
MAMP now works on Windows.  
[MAMP](https://www.mamp.info/en/mac/) is a free, local server environment that can be installed under macOS and Windows with just a few clicks. MAMP provides them with all the tools they need.

## Natively
You can install each tool by yourself on your Windows machine. You will have to find a guide that installs IIS, PHP8, and a supported database.
